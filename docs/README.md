## 🎯 このテンプレートの使い方

1. **プロジェクト初期化**

   - リポジトリをクローンまたはテンプレートとして使用
   - 依存関係のインストール: `pnpm install`
   - 環境変数の設定

2. **開発開始**

   - 開発サーバー起動: `pnpm dev`
   - コードの変更とホットリロード

3. **カスタマイズ**
   - プロジェクト固有のドキュメントを追加
   - 必要に応じてアーキテクチャを調整

## 📝 ドキュメント作成のガイドライン

- **明確で簡潔に**: 技術的な内容を分かりやすく記述
- **実例を含める**: コードサンプルや設定例を提供
- **最新に保つ**: コードの変更に合わせてドキュメントも更新
- **図表を活用**: 複雑な概念は図で説明
